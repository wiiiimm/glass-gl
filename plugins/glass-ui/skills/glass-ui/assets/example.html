Glass UI demo

Frosted card

This panel is class="glass". It blurs and saturates the colorful gradient behind it, with a bright rim and an inner top highlight.

Dark glass

class="glass glass-dark" — tints with black instead of white, for placing over lighter areas.

Refraction (tier 2)

class="glass-refract" — uses an SVG feDisplacementMap so the gradient behind it bends, not just blurs. No WebGL. (Safari falls back to plain blur.)
